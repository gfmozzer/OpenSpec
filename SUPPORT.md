# Suporte

Use o canal certo para cada tipo de assunto:

- Bug reproduzível: abra uma issue com contexto, versão e passos de reprodução.
- Proposta de melhoria: abra uma issue de feature ou discussão de design.
- Dúvida de uso: abra uma discussão ou issue de pergunta, se o repositório usar esse fluxo.
- Vulnerabilidade: siga [SECURITY.md](SECURITY.md) e reporte em privado.
- Conduta/comunidade: use os canais de manutenção indicados no repositório e consulte [CODE_OF_CONDUCT.md](CODE_OF_CONDUCT.md).

## Antes de pedir ajuda

- Confira o `README.md`.
- Verifique a documentação em `docs/`.
- Revise o fluxo SDD em `AGENTS.md` e `.sdd/AGENT.md`.
- Informe sistema operacional, versão do Node, versão do pacote e comando executado.
